Push button behavior and pull request actions

Push: Sends your latest committed changes from the local branch to the remote repository so others can see and review them.

Create PR (Draft): Opens a new pull request in draft mode. Draft PRs share your branch for early feedback without signaling readiness to merge; reviewers can comment, but the PR must be marked "Ready for review" before merging.

Create PR: Opens a pull request in ready-for-review mode. This starts the formal review/merge process immediately, allows reviewers to approve, and lets maintainers merge once checks pass.
